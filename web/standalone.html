Contrôle ta trottinette Ninebot / Xiaomi en Bluetooth

Bluetooth web non disponible dans ce navigateur.

Sur iPhone, ouvre cette page dans Bluefy (navigateur gratuit sur l'App Store) : Safari ne gère pas le Bluetooth.

1. Allume ta trottinette et reste à côté.
2. Appuie sur « Chercher ma trottinette ».
3. Elle apparaît sous un nom comme MIScooter…, Ninebot… ou M365…. Sélectionne-la.

Si rien n'apparaît : réveille la trottinette (appuie sur son bouton) puis réessaie, ou utilise « Afficher tous les appareils ». Les lignes avec un code (ex. 7068F751-…) sont d'autres objets Bluetooth autour de toi, pas ta trottinette.

Débridage

🛴 Ma trottinette est une Ninebot E-series (E22 / E25 / E45 / E45E)
Ces modèles sont chiffrés : cette app (et toute app iPhone) ne peut pas les débrider. La seule méthode qui marche passe par ScooterHacking Utility, une app Android gratuite (emprunte un téléphone Android).

Vitesse max réelle une fois débridée : ~29–30 km/h sur E45 (le « 45 » n'est PAS la vitesse). Le matériel ne peut pas faire 45 km/h.

Ton firmware est DRV 2.7.2 → le changement de série est bloqué depuis la 2.7.1. Il faut d'abord downgrader :
1. Installe SHU : utility.cfw.sh/apk (sur Android).
2. Dans SHU : Load Original → DRV → DRV271 (version modifiée qui autorise le downgrade) → FLASH.
3. Puis Load Original → DRV → DRV255 → FLASH.
4. Bouton « Change Region » → choisis le preset E45 30 km/h.
5. Laisse redémarrer, passe en mode Sport (S) → ~30 km/h.

Guide complet illustré : scooterhack.in/eseriesunlock

⚠️ Flasher un firmware comporte un petit risque. Suis le guide à la lettre, batterie chargée, et n'utilise pas de CFW esx.cfw.sh avant (ça bloque cette méthode).
Ça ne marche pas ? → Solutions
Avant tout : trottinette allumée, à l'arrêt, sur béquille, pas en charge, et reste à moins d'1 m.

1. Appuie sur « Déconnexion » puis reconnecte-toi et relance le débridage.
2. Change d'abord la région sur 🌍 Global (section Région), puis règle la vitesse.
3. Réessaie 2–3 fois : certains firmwares n'acceptent l'écriture qu'après plusieurs tentatives.
4. Si l'app dit « commande acceptée » mais l'écran reste à 25 : c'est un bridage firmware (limite légale). Aller au-delà exige un firmware custom (via un vrai outil sur ordinateur type ScooterHacking Utility) — impossible et risqué depuis une app web, donc non proposé ici.
5. Si l'app dit « ne répond pas » : ton modèle est chiffré (Max G30 / F / G2…) et n'est pas géré par cette version web.
Retirer le limiteur peut rendre la trottinette non homologuée pour la voie publique et annuler la garantie. Modèles chiffrés récents (Max G30+) non pris en charge par cette version web.
